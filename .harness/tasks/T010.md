# T010 ‚Äî Shared visual primitives: sweep wedge, pulsing dot, radar mark

## Overview

The three animated primitives reused across screens: the rotating angular-gradient sweep wedge, the
pulsing status dot, and the concentric-ring radar mark (splash + consent variants). Built as isolated
leaf views so the measuring screen's ~4 Hz state churn can never reset their repeat-forever animations.

## Do

All in `Sources/DesignSystem/`. Reference design: `design/design_handoff_scout/README.md` ¬ß1‚Äì¬ß2.

1. `SweepWedgeView.swift`:

```swift
struct SweepWedgeView: View {
    init(color: Color, opacity: Double, wedgeDegrees: Double, period: Double, rotating: Bool)
}
```

   - A circle filled with an `AngularGradient` built from STOPS over the full turn (an
     endAngle < 360¬∞ does NOT leave the rest transparent ‚Äî it clamps to the nearest stop):
     `[.init(color: color.opacity(0), location: 0), .init(color: color, location: wedgeDegrees/360),
     .init(color: color.opacity(0), location: wedgeDegrees/360 + 0.0001), .init(color:
     color.opacity(0), location: 1)]`. Fade to `color.opacity(0)`, NEVER `.clear` (transparent black
     fringes gray). Apply `opacity` to the whole view.
   - Rotation: `@State private var isRotating = false`, `.rotationEffect(.degrees(isRotating ? 360 :
     0))`, `.animation(.linear(duration: period).repeatForever(autoreverses: false), value:
     isRotating)`, `.onAppear { isRotating = true }`. This keyed-value idiom is load-bearing: never
     use un-keyed `.animation`, never `withAnimation` in `onAppear`, and keep this view's inputs
     stable so parent re-renders can't reset it.
   - `rotating: false` OR `@Environment(\.accessibilityReduceMotion) == true` ‚Üí no rotation (static
     wedge, no animation applied).

2. `PulsingDot.swift`:

```swift
struct PulsingDot: View { init(color: Color, diameter: CGFloat = 7) }
```

   - Circle; pulse per handoff (`0.3 opacity/0.8 scale ‚Üí 1/1 ‚Üí back`, full cycle
     `ScoutMotion.pulsePeriod`): `.animation(.easeInOut(duration: ScoutMotion.pulsePeriod / 2)
     .repeatForever(autoreverses: true), value: pulsing)` with the same keyed `onAppear` idiom.
   - Reduce Motion ‚Üí static at full opacity/scale.

3. `RadarMarkView.swift`:

```swift
struct RadarMarkView: View {
    enum Style { case splash, consent }
    init(style: Style)
}
```

   - `.splash` (196√ó196 pt): three stroked circles r 92/62/32 pt, stroke 1.5 pt, `ScoutTheme.white`
