# T015 — Paused states: No-cellular fallback + consent-declined idle

## Overview

Full-fidelity rendering of the two non-measuring variants of the home screen. "No cellular" is
specced in the handoff (`design/design_handoff_scout/README.md` §4); the consent-declined idle state
is a deliberate, owner-approved addition (iOS apps must not self-exit, so "Not now" lands here) that
reuses the same calm structure plus a Start button.

## Do

Extend `Sources/Screens/MeasuringView.swift` (or a sibling file in `Sources/Screens/`) so the paused
branches render fully:

Variant A — **No cellular** (consent given, `!session.cellularAvailable`):
- Status row: STATIC dot 7 pt at `ScoutTheme.white(0.28)` (no pulse) + "PAUSED" — 14 pt semibold,
  tracking 2.5, `ScoutTheme.white(0.4)`.
- Dial: `SweepDialView(mode: .idle)`.
- Footer (id `paused.title` on the title): "No cellular to measure" — 20 pt semibold,
  `ScoutTheme.white(0.82)`; body (id `paused.body`): "Scout measures your cellular speed — download
  and upload. Turn off Airplane Mode or Wi-Fi to start sweeping." — 15 pt, `ScoutTheme.white(0.5)`,
  centered, max width 280 pt, with the word "cellular" emphasized (semibold, `ScoutTheme.white(0.7)`).
- Resumes AUTOMATICALLY when the session's `cellularAvailable` flips back (session handles it —
  no button here).

Variant B — **Idle / sweeping off** (consent NOT given — the "Not now" path):
- Same status row ("PAUSED") and idle dial.
- Footer title (id `paused.title`): "Sweeping is off" — 20 pt semibold, `ScoutTheme.white(0.82)`.
- Body (id `paused.body`): "Scout only measures cellular data when you choose. Start sweeping to
  measure this spot." — 15 pt, `ScoutTheme.white(0.5)`, centered, max width 280 pt.
- Button (id `paused.startButton`), 20 pt below body: "Start sweeping" — width ≤ 240 pt, height 50,
  corner radius 14, fill `ScoutTheme.great`, label 17 pt semibold `ScoutTheme.onAccent`. Triggers the
  SAME grant action as the consent screen's primary button (router `startSweeping()` → persists
  consent → session starts).
- No data transfer may occur in this state (the session was never started — verify nothing calls
  `session.start()` while consent is false; the P0/scenePhase guard from the app shell already
  encodes this, keep it intact).

Remove the leftover stub paused texts once both variants render.
