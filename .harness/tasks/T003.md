# T003 — Design tokens: ScoutTheme (colors + motion constants)

## Overview

Create the single source of truth for Scout's design tokens — the exact colors and motion timings from
the design handoff (`design/design_handoff_scout/README.md`, "Design tokens" section) — so every later
UI task references named constants instead of re-deriving values.

## Do

Create `Sources/DesignSystem/ScoutTheme.swift` containing:

1. A `Color` hex initializer (fileprivate or internal): `Color(scoutHex: UInt32)` producing sRGB colors,
   e.g. `Color(scoutHex: 0x63D99B)`.
2. `enum ScoutTheme` (namespace, no cases) with these EXACT values (hex already converted from the
   handoff's OKLCH — do not re-convert):
   - `static let background = Color(scoutHex: 0x050506)`
   - `static let great = Color(scoutHex: 0x63D99B)`      // oklch(0.80 0.14 158) — also the brand accent
   - `static let usable = Color(scoutHex: 0xEEBB58)`     // oklch(0.82 0.13 82)
   - `static let poor = Color(scoutHex: 0xD58679)`       // oklch(0.70 0.10 30)
   - `static let uploadArc = Color(scoutHex: 0x78B3D6)`  // oklch(0.74 0.08 235) — inner dial arc
   - `static let uploadText = Color(scoutHex: 0x99C5DF)` // oklch(0.80 0.06 235) — upload numerals
   - `static let onAccent = Color(scoutHex: 0x04140C)`   // text on the green primary button
   - `static func white(_ opacity: Double) -> Color`     // Color.white.opacity(n) helper for the
     handoff's white-opacity text scale (1.0/0.82/0.72/0.60/0.55/0.50/0.40/0.35/0.30/0.22 used per-view)
3. `enum ScoutMotion` with the handoff's timing constants:
   - `static let splashSweepPeriod: Double = 3.2`   // splash radar sweep rotation
   - `static let dialSweepPeriod: Double = 3.4`     // measuring dial sweep rotation
   - `static let pulsePeriod: Double = 1.6`         // status-dot pulse (full cycle)
   - `static let arcDuration: Double = 0.28`        // arc-length transition (linear)
   - `static let colorDuration: Double = 0.45`      // quality-color transition (ease)
   - `static let tickInterval: Double = 0.25`       // ~4 Hz reading cadence

Doc-comment each constant with what it's for. No views in this task — constants only.

## Done when

- `Sources/DesignSystem/ScoutTheme.swift` exists with ALL constants above at exactly these values and
  names (later tasks bind to these names verbatim).
- The project builds via the standard DoD command; SwiftFormat/SwiftLint clean.
