# T030 — RadarMarkView (splash + consent variants), shown on the splash stub

## Overview

The concentric-ring radar mark — Scout's brand mark — in its two design variants, composed from the
already-built SweepWedgeView, and placed onto the splash stub so it's visible in the running app
(and in the harness's screenshot check).

## Do

1. `Sources/DesignSystem/RadarMarkView.swift`:

```swift
struct RadarMarkView: View {
    enum Style { case splash, consent }
    init(style: Style)
}
```

   Per `design/design_handoff_scout/README.md` §1–§2:
   - `.splash` (196×196 pt): three stroked circles r 92/62/32 pt, stroke 1.5 pt, `ScoutTheme.white`
     at 0.09/0.11/0.13 (outer→inner); center dot r 6 pt filled `ScoutTheme.great`; BEHIND the rings
     a `SweepWedgeView(color: ScoutTheme.great, opacity: 0.18, wedgeDegrees: 98, period:
     ScoutMotion.splashSweepPeriod, rotating: true)` sized 196.
   - `.consent` (76×76 pt): two rings r 35/22 pt stroke 1.5 pt white 0.10/0.12; dot r 4.5; wedge
     92°, opacity 0.20, `rotating: false` (static).
   - All radii are FIXED design pt (explicit frames, no GeometryReader).
2. Update `SplashStubView` to display `RadarMarkView(style: .splash)` centered above its existing
   "Scout" text, on `ScoutTheme.background`.
