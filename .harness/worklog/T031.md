2. **DialContent.swift** — Shared value type for dial + screens
   ```swift
   struct DialContent: Equatable {
       let downDisplay: String
       let upDisplay: String
       let downFraction: Double
       let upFraction: Double
       let qualityColor: Color
       let generationText: String
   }
   ```
   - Added `SignalQuality.color` extension mapping `.great` → `ScoutTheme.great`, `.usable` → `ScoutTheme.usable`, `.poor` → `ScoutTheme.poor`

3. **DialCenterStack.swift** — Center stack with live & idle modes
   - **Live mode**: Generation label (15 pt, medium, tracking 2, white 0.5, hidden when empty)
     → Hero download (80 pt, semibold, tracking -3, monospacedDigit, quality color, animated on color change)
     → "↓ Mbps down" row (16 pt medium, white 0.55)
     → "↑ n.n up" row (19 pt semibold upload value in blue, "up" label 14 pt)
   - **Idle mode**: "—" em dash (96 pt, semibold, white 0.22) over "Mbps" (18 pt medium, white 0.35)
   - Hero text uses `@ScaledMetric(relativeTo: .largeTitle)` for Dynamic Type
   - Color animation on hero: `.animation(.easeInOut(duration: ScoutMotion.colorDuration), value: color)`
   - `.minimumScaleFactor(0.6)` + `.lineLimit(1)` on hero to compress instead of clip

4. **MeasuringStubView.swift** — Integrated DialCenterStack

   - Paused branch: shows "PAUSED" + `DialCenterStack(mode: .idle)` + conditionally "Start sweeping" button
   - `measuring.hero` identifier preserved on center stack's hero
