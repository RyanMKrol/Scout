# T010 — Animated primitives: SweepWedgeView + PulsingDot

## 2026-07-17

**Done.** Built both animated leaf views with keyed-value animation patterns to isolate rotation and pulse from parent state churn.

### What was done

1. **SweepWedgeView.swift** (Sources/DesignSystem/):
   - Angular-gradient sweep wedge with parameterized color, opacity, wedge degrees, and rotation period
   - Gradient stops: transparent → color → transparent + transparent at end, as spec'd
   - Rotation via keyed-value `.animation(value: isRotating)` with `.linear().repeatForever()` 
   - Reduce Motion support: skips animation setup if `accessibilityReduceMotion == true`
   - Preview on ScoutTheme.background

2. **PulsingDot.swift** (Sources/DesignSystem/):
   - Circle pulsing via opacity (0.3 → 1 → 0.3) and scale (0.8 → 1 → 0.8) 
   - Keyed-value animation `.easeInOut(duration: ScoutMotion.pulsePeriod / 2).repeatForever(autoreverses: true)`
   - Reduce Motion support: static at full opacity/scale when enabled
   - Parameterized color and diameter (default 7pt)
   - Preview on ScoutTheme.background
